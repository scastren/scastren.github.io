# Markdown-komennot ja niiden selitykset

## 1. Otsikot
```markdown
# Otsikko 1
## Otsikko 2
### Otsikko 3
```
**Selitys:** `#` määrittää otsikon tason. Yksi risuaita = suurin otsikko, lisää risuaitoja = pienempi otsikko.

## 2. Lihavointi
```markdown
**Lihavoitu teksti**
```
**Selitys:** Kahden tähden väliin kirjoitettu teksti näkyy **lihavoituna**.

## 3. Kursivointi
```markdown
*Kursivoitu teksti*
```
**Selitys:** Yhden tähden väliin kirjoitettu teksti näkyy *kursivoituna*.

## 4. Lainaus
```markdown
> Tämä on lainaus
```
**Selitys:** `>` tekee tekstistä lainauksen (blockquote).

## 5. Luettelot
```markdown
- Ensimmäinen kohta
- Toinen kohta
```
**Selitys:** `-` tai `*` aloittaa luettelon. Käytetään järjestämättömiin listauksiin.

## 6. Numeroitu lista
```markdown
1. Ensimmäinen
2. Toinen
```
**Selitys:** Numero + piste tekee järjestetyn listan.

## 7. Linkki
```markdown
[Linkin teksti](https://www.esimerkki.fi)
```
**Selitys:** `[teksti](URL)` luo klikattavan linkin.

## 8. Kuva
```markdown
![Alt-teksti](kuva.jpg)
```
**Selitys:** `![alt-teksti](kuvan URL)` lisää kuvan.

## 9. Koodi
```markdown
`inline-koodi`
```
**Selitys:** Yksi backtick (`) tekee koodin rivin sisään.

## 10. Koodilohko
```markdown
```
Monirivinen
koodilohko
```
```
**Selitys:** Kolme backtickiä ennen ja jälkeen tekee monirivisen koodilohkon.

## 11. Vaakaviiva
```markdown
---
```
**Selitys:** Kolme viivaa tekee vaakasuoran erottimen.

## 12. Taulukko
```markdown
| Sarake 1 | Sarake 2 |
|----------|----------|
| Arvo 1   | Arvo 2   |
```
**Selitys:** `|` ja `-` muodostavat taulukon.
